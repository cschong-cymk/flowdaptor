How we helped Sagelion make The Jade Tithe

The team at Sagelion had a story they wanted to tell. A fantasy drama. Original characters, a whole world, cinematic visuals. The kind of thing that would normally need a full production budget and a team of twenty.

They came to us with the concept. We helped them figure out the rest.

The project became The Jade Tithe — and the first thing we had to solve was consistency. AI video generation is easy. Getting the same character to look like herself across 70 clips is a completely different problem. So we built a pipeline around that — locking character references before anything else got generated, then working scene by scene from there.

We handled the technical side: setting up the workflow in OpenClaw, structuring the production phases, making sure everything fed correctly so nothing drifted. Sagelion brought the creative vision. We made sure the tools kept up with it.

The first output was a music video. See for yourself.
